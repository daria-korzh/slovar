# slovar
есть словарь русского языка состоящий из не менее чем из тысячи слов при этом слова отсортированы в лексикографическом порядке пользователь этого словаря хочет проверить правильно ли он пишет слова на русском языке и вводит с клавиатуры используя диалоговое окно кьюти то слово которое ему интересно ваше произведение принимает это слово и методом бинарного поиска определяет 1 есть ли это слово в словаре и 2 есть ли слова близкие по написанию,и если такие слова есть то пользователю сообщается правильный вариант например: слово "зума" а в словаре есть слова "кума" "зука" "зима" соответственно в качестве близкого по значению слова будет "кума"
